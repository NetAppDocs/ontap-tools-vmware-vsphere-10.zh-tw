= 設定預留位置資料存放區
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
設定佔位資料儲存以在復原站點的 vCenter 清單中為受保護的虛擬機器 (VM) 保留空間。佔位資料儲存需要最小容量，因為佔位虛擬機器很小，通常只使用幾百千位元組。

.開始之前
* 確保受保護站點和復原站點已連線。
* 驗證資源映射是否已配置。


.步驟
. 登入 vCenter Server 並選取 * 站台恢復 * > * 站台 * 。
. 選取您受保護的網站，然後選取 * 管理 * 。
. 在管理索引標籤中選取 * 預留位置資料存放區 * > * 新增 * ，以建立新的預留位置資料存放區。
. 選取適當的資料存放區，然後選取 * 確定 * 。
+

NOTE: 佔位資料儲存可能駐留在本機或遠端儲存上，但它們不需要複製。

. 重複步驟 3 至 5 、為恢復站台設定預留位置資料存放區。
