= 管理儲存臨界值
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以將臨界值設定為當磁碟區和集合容量達到特定層級時、在 vCenter Server 中接收通知。

.步驟：
. 登入 vSphere 用戶端。
. 在捷徑頁面中，選取外掛程式區段下方的 * NetApp ONTAP tools* 。
. 在ONTAP工具的左側窗格中，前往 *設定* > *閾值設定* > *編輯*。
. 在「*編輯閾值*」視窗中，在「*接近滿*」和「*滿*」欄位中提供所需的值，然後選擇「*儲存*」。您可以將閾值恢復為建議的預設值：接近滿格為 80，滿格為 90。
