= 在其他節點上啟動 SSH
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
升級之前、您需要在其他節點上啟動 SSH 。

.開始之前
為您的 vCenter Server 執行個體啟用 VASA Provider 擴充功能。

.關於這項工作
升級前，請對每個節點重複此步驟。

.步驟
. 從vCenter Server開啟主控台至VASA Provider。
. 以維護使用者身分登入。
. 輸入 `4` 選擇 Support and Diagnostics 。
. 輸入 `1` 以選取 Access 診斷 Shell 。
. 輸入 `y` 以繼續。
. 執行命令 _Sudo systemctl restart ssh_ 。
