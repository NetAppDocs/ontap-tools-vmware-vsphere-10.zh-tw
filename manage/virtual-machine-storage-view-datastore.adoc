= 虛擬機器儲存檢視
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
儲存視圖顯示虛擬機器建立的vVols清單。


NOTE: 此視圖適用於至少有一個來自ONTAP tools for VMware vSphere管理的vVols資料儲存的磁碟的虛擬機器。

.步驟
. 從 vSphere Client 轉到虛擬機器。
. 選取右窗格中的 * 監控 * 標籤。
. 選擇 * NetApp ONTAP tools* > * Storage* 。「儲存 * 」詳細資料會顯示在右窗格中。您可以看到 VM 上存在的 vVols 清單。
+
您可以使用「管理欄」選項來隱藏或顯示不同的欄。
