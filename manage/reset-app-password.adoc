= 重設應用程式使用者密碼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用ONTAP tools for VMware vSphere，請依照下列步驟重設 SRA 和 VASA Provider 在 vCenter Server 上註冊所需的應用程式使用者密碼。

.步驟
. 開啟網頁瀏覽器並導航至： `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. 使用在ONTAP工具部署期間設定的管理員憑證登入。
. 從側邊欄中選擇“設定”。
. 在 *VASA/SRA 憑證* 頁面上，選擇 *重設密碼*。
. 請輸入並確認新密碼。
. 選擇“重設”以套用新密碼。
