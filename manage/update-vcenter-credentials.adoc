= 在 ONTAP 工具中更新 vCenter Server 認證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用維護控制台更新 vCenter Server 執行個體憑證。

.開始之前
您必須擁有維護使用者登入認證。

.關於這項工作
如果部署後變更了 vCenter Server 憑證，請使用此程序更新它們。

.步驟
. 從vCenter Server開啟主控台至VASA Provider。
. 以維護使用者身分登入。
. 輸入 `2` 以選取系統組態功能表。
. 進入 `8` 變更 vCenter 憑證。
