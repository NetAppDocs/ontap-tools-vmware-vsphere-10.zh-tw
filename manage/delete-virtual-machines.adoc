= Vasa 清理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要完成 VASA 清理流程，請依照下列步驟操作。


NOTE: 建議在開始 VASA 清理之前刪除所有vVols資料儲存。

.步驟
. 移至 \https://OTV_IP:8143/Register.html 以取消登錄外掛程式
. 確認 vCenter Server 上不再提供外掛程式。
. 關閉適用於 VMware vSphere VM 的 ONTAP 工具。
. 刪除適用於 VMware vSphere VM 的 ONTAP 工具。
